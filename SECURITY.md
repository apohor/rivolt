# Security

If you discover a security vulnerability in Rivolt, please report it privately.

## Reporting

- **Email:** pogiant@gmail.com (subject line prefix: `[rivolt-security]`)
- **Do not** open a public GitHub issue for security reports.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

I'll acknowledge within 72 hours and aim to ship a fix within 14 days for confirmed issues.

## Scope

In scope:
- Rivolt source code in this repository
- Official Docker images published from this repo

Out of scope:
- The Rivian API itself
- Third-party integrations (Enphase, Powerwall, etc.)
- Self-hosted deployments where the operator misconfigured their own network / TLS

## Disclosure

After a fix ships, I'll credit the reporter in the release notes unless they prefer to remain anonymous.
